Как работает программа

Сначала программа задаёт пользователю вопросы:

сколько паролей нужно создать
какой длины должен быть пароль
использовать ли цифры
использовать ли заглавные буквы
использовать ли строчные буквы
использовать ли специальные символы
нужно ли убрать неоднозначные символы (например l и 1, O и 0)

После этого формируется строка допустимых символов.
Если пользователь исключил все группы символов, программа завершится с сообщением об ошибке.
Далее создаётся нужное количество паролей заданной длины с помощью случайного выбора символов.

Что поддерживается
цифры 0–9
строчные буквы
заглавные буквы
специальные символы !#$%&*+-=?@^_
возможность убрать похожие символы (il1Lo0O), чтобы избежать путаницы

Пример работы
Пользователь указывает:
3 пароля
длина 12
включить цифры и буквы
убрать неоднозначные символы
Программа выводит, например:

aG7kT9qP2zLm
Q4sJ8hTrY5dK
mN3xV7pRbW2c
